Функция Фибонначи: Временная сложность - O(2^n). Для оптимизации можно использовать итеративный подход, сложность которого будет O(n)
Функция Аккермана: Временная сложность функции Аккермана растёт как 2^2^...^2 — степенная башня из m двоек. Для оптимизации можно использовать мемоизацию для хранения результатов ранее выполненных вызовов функции Аккермана, сложность будет O(n*m)
Функция поиска числа: Временная сложность - O(n*m). Для оптимизации можно  испольовать итеративный подход, сложность которого будет идентична рекурсивному, но функция будет выполняться быстрее за счёт отсутствия рекурсивных вызовов
